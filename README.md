**Sistema de Vendas com Procedure (PostgreSQL)**

🎯 Objetivo
- Criação tabelas de produtos e vendas
- Relacionar os dados
- Realizar uma venda com validações

⚙️ Regras de Negócio:
- Verificar se o produto existe
- Verificar se há estoque suficiente
- Calcular o valor total (preco * quantidade)
- Inserir a venda na tabela vendas
- Atualizar o estoque do produto
- Caso não haja estoque, exibir mensagem de erro
